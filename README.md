# 建筑生模拟器：从硫酸纸到大师

真实还原建筑系学生五年学制（120周）的"修仙"历程的文字模拟器游戏。

## 游戏特色

- **16种身份组合**：4种学校背景 × 4种家庭背景
- **25个专业课题**：覆盖大一到大五的完整课程体系
- **70条事件内容**：50条随机事件 + 20条交互抉择
- **专业评图系统**：基于进度和质量的动态评分
- **多维度结局**：从普利兹克奖到退学的多种结局
- **工业设计风格UI**：极简现代的界面设计

## 如何运行

### 开发模式
```bash
npm install  # 首次运行需要安装依赖
npm run dev  # 启动开发服务器
```

访问 http://localhost:5173 开始游戏

### 生产构建
```bash
npm run build  # 构建生产版本
```

构建产物在 `dist` 目录，可以部署到任何静态服务器。

## 游戏玩法

1. **初始化**：随机抽取身份（学校+家庭背景）
2. **每周行动**：
   - 红牛续命：增加进度，提升压力
   - 死磕逻辑：提升质量，增加设计能力
   - B站大学：学习软件技能
   - 大师讲座：提升设计感
   - 天台吹风/火锅整顿：减压
   - 接私活：赚钱但增加压力
   - 使用专属技能（每月一次）
3. **评图机制**：
   - 期中评图（第6周）：进度需≥50%
   - 期末评图（第12周）：进度需≥100%
   - 质量评级：E/C/B/A
4. **模型制作**：第5周和第11周制作模型
5. **结局触发**：
   - 完成120周后根据属性判定正常结局
   - 破产/崩溃/劝退会提前触发失败结局

## 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **状态管理**：React Context + useReducer
- **样式方案**：Vanilla CSS
